# van-nav：自定义导航

> github：[https://github.com/mereithhh/van-nav](https://github.com/mereithhh/van-nav)

启动服务

```shell
docker-compose up -d
```

停止服务

```shell
docker-compose down
```

浏览器访问：

- 前台：[http://127.0.0.1:6412/](http://127.0.0.1:6412/)
- 后台：[http://127.0.0.1:6412/admin/](http://127.0.0.1:6412/admin/)，默认帐密 admin/admin
